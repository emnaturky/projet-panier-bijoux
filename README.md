# projet-panier-bijoux
Panier Bijoux - Site E-commerce de Bijoux

Un mini site e-commerce de bijoux élégants développé en **PHP 8 + MySQL** avec gestion de panier via sessions.

---

##  Fonctionnalités

- Affichage des produits avec images et catégories
- Ajout d'articles au panier
- Mise à jour des quantités dans le panier
- Suppression d'articles
- Calcul automatique du total
- Validation de commande (enregistrement en base de données)
- Messages de confirmation
- Design responsive et élégant (Bootstrap + style or)

---

## Technologies utilisées

- **PHP** (sessions, PDO)
- **MySQL** / MariaDB
- **Bootstrap 5** (frontend)
- **HTML5 & CSS3**
